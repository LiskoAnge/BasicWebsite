AUTODESK MAYA SCENE

The 3D work I excecuted on Maya represents the first scene of the storyboard.
I built the scene with the help of several Polygon Primitives and few Nurbs.
I worked mainly with the “Poly Modeling” tools, but I also began experimenting with “Sculpting” and “Curves/Surfaces”.
I textured my creation with images found on Google, making sure they are free to use and modify, setting the option on the advanced research.
© 2018, Screenshot Angela Dematte.
On BlueJ I had the opportunity to develop my programming abilities with the Java language. I have been working on the creation of the 1976 game "Breakout". I recreated the bat, ball and bricks with different colours. My code was composed of new methods, loops, variables, but I also took advantage of the GUI system; for example, I added three tries (or “lives”) before the players’ failure of the game and managed to communicate it with a Label, which I utilized in the “game-over” sign too. Moreover, I connected one of the keyboard’s buttons to my code, calling it the “restart button” to refresh the score and the game objects in order to play all over again once the game ends.
